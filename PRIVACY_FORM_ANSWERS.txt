------------------------------------------------------
1. SINGLE PURPOSE DESCRIPTION
------------------------------------------------------
This extension adds a "More Examples" button to Readlang flashcard practice pages that generates additional example sentences using AI, helping users learn vocabulary words in context.

------------------------------------------------------
2. HOST PERMISSIONS JUSTIFICATION
------------------------------------------------------
Host permissions are required to make API calls to AI providers (Mistral AI, OpenAI, Anthropic Claude, Google Gemini, and OpenRouter) to generate example sentences. The extension sends the vocabulary word being studied to the user's selected AI provider and displays the generated examples on the page. No data is collected or sent elsewhere.

------------------------------------------------------
3. STORAGE PERMISSION JUSTIFICATION
------------------------------------------------------
Storage permission is required to save the user's settings locally, including their selected AI provider, API key, and optional model preference. This data is stored only on the user's device using Chrome's built-in storage API and is never sent to any external server.

------------------------------------------------------
4. REMOTE CODE JUSTIFICATION
------------------------------------------------------
This extension does not use remote code. All JavaScript is bundled within the extension package. The only external requests are API calls to AI providers (selected by the user) to generate example sentences, which return text data only, not executable code.

------------------------------------------------------
